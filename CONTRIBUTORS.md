# Contributors

Recursive Tarot is released under **CC-BY-SA-4.0** — the "share-alike" half of that licence is a
promise: this library stays open, and everyone who builds it stays credited. Every merged
contribution's author already rides along in the git history (`git log`); this file just makes
that list easy to find without cloning the repo.

The list below is young — right now it's essentially one builder plus one early collaborator. If
you fix a card, add a deck, write a course, or improve the site and get a **pull request merged**,
add yourself here in that same PR (or ask the maintainer to) and you're on the list. No invented
names, no false crowd — just whoever has actually merged work.

## How to get listed

1. Open a pull request against `dev` — a card correction, a new deck, a course, a fix to the site.
2. Once it's merged, your authorship is permanent in the git history — that's the real record.
3. Add your name to the **Contributors** list above in the same PR (or a follow-up one) so it's
   visible here too, not just buried in `git log`.

That's the whole process. Small fixes count as much as big ones — the commons grows one merged PR
at a time.
